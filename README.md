# Jump Server Essentials

JSE is a set of modular plugins for enabling standard features present in most Team Fortress 2 jump servers.

## Modules
* Autosave
* Core
* Foresight
* JumpBOT
* Marker
* Map Info
* Scoreboard
* Show Keys
* Teleport
* Tracker

## Required
* Sourcemod
